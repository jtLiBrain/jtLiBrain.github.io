for推导式 for循环
翻译前 翻译后 翻译前 翻译后
一个生成器 for (p <- e) yield e′ e.map { case p => e′ } for (p <- e) yield e′ e.foreach { case p => e′ }
两个生成器 for (p <- e; p′ <- e′; …) yield e″ e.flatMap { case p => for (p′ <- e′; …) yield e″ } for (p <- e; p′ <- e′; …) yield e″ e.foreach { case p => for (p′ <- e′; …) e″ }
